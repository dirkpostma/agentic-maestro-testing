# Agentic Maestro Testing POC

**Goal**: Verify if AI agents can autonomously run E2E tests, detect failures, and fix issues.

## The Experiment

Can an AI agent:

1. Set up and run Maestro E2E tests
2. Detect test failures
3. Analyze and fix code issues
4. Re-run until all tests pass

## For AI Agents

See [AGENT_INSTRUCTIONS.md](./AGENT_INSTRUCTIONS.md) for complete setup and testing procedures.

## The App

Simple counter app with:

- Increment button
- Decrement button
